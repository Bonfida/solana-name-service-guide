# Vue SDK

The package `@bonfida/sns-vue` contains a set of useful Vue composables to help you integrate SNS into your dApp. A fully working Vue app can be found on [Github](https://github.com/SolanaNameService/sns-sdk/tree/main/examples/vue-app) with a working example for each of the composable:

- [useDomainOwner](https://github.com/SolanaNameService/sns-sdk/blob/main/examples/vue-app/src/components/examples/useDomainOwner.vue)
- [useDomainSize](https://github.com/SolanaNameService/sns-sdk/blob/main/examples/vue-app/src/components/examples/useDomainSize.vue)
- [useDomainsForOwner](https://github.com/SolanaNameService/sns-sdk/blob/main/examples/vue-app/src/components/examples/useDomainsForOwner.vue)
- [usePrimaryDomain](https://github.com/SolanaNameService/sns-sdk/blob/main/examples/vue-app/src/components/examples/useFavoriteDomain.vue)
- [useProfilePic](https://github.com/SolanaNameService/sns-sdk/blob/main/examples/vue-app/src/components/examples/useProfilePic.vue)
- [useRecords](https://github.com/SolanaNameService/sns-sdk/blob/main/examples/vue-app/src/components/examples/useRecords.vue)
- [useReverseLookup](https://github.com/SolanaNameService/sns-sdk/blob/main/examples/vue-app/src/components/examples/useReverseLookup.vue)
- [useSubdomains](https://github.com/SolanaNameService/sns-sdk/blob/main/examples/vue-app/src/components/examples/useSubdomains.vue)
